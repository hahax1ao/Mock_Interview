# 电子信息保研模拟面试 Agent V1 设计

## 目标

构建一个自用的本地网页应用，通过阿里云百炼实时语音模型模拟四人复试小组，并在面试结束后提供有证据的评分、示范回答、七天训练计划和长期趋势。

## 已确认边界

- 聚焦电子信息工程及相近信息工科。
- 使用个人材料、目标院校材料和专业参考资料，不联网检索。
- 支持10、20和30分钟面试；默认20分钟均衡综合面。
- 四角色为主考官、专业基础老师、科研项目导师和英语老师，共用一个声线。
- 面试中不反馈；结束后按照固定量表复盘。
- 文件、索引、转写和报告保存在本机；不保存原始录音。
- 云端模型切换为阿里云百炼按量付费服务。

## 评分

总分100分：专业基础25、项目科研25、逻辑表达15、英语交流15、真实性10、抗压表现10。每项评分必须包含原问题、回答证据、问题类型、解释、改进建议、材料来源和置信度。

## 隐私

原始材料与索引留在本机，仅向百炼发送当前任务所需的最小文本片段。API Key 仅存在被忽略的本地环境文件或进程环境中。
